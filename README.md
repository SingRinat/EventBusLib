# EventsBus

High-performance Event Bus для .NET приложений с поддержкой синхронных/асинхронных подписок, слабых ссылок и оптимизацией памяти.

## Features

- ✅ **Синхронные и асинхронные обработчики**
- ✅ **Слабые ссылки** - автоматическая очистка памяти
- ✅ **Приоритеты подписок**
- ✅ **Фильтрация событий**
- ✅ **Паттерн Request/Response** (как Mediator)
- ✅ **Пакетная публикация событий**
- ✅ **Метрики и мониторинг**
- ✅ **Health checks**
- ✅ **DI Ready** - простая интеграция
- ✅ **Оптимизация памяти** - Object pooling
- ✅ **Thread-safe**
